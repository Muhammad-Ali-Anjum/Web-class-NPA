Transform

Transform is a CSS property that allows you to apply various transformations to an element, such as scaling, rotating, skewing, and translating. It can be used to create dynamic and visually appealing effects on web pages.

Types of Transformations

  • Scale: Resizes an element. For example, transform: scale(1.5); will make the element 1.5 times larger.
  • Rotate: Rotates an element by a specified angle. For example, transform: rotate(45deg); will rotate the element 45 degrees clockwise.
  • Skew: Skews an element along the X or Y axis. For example, transform: skew(20deg); will skew the element by 20 degrees.
  • Translate: Moves an element from its original position. For example, transform: translate(50px, 100px); will move the element 50 pixels to the right and 100 pixels down.

Combining Transformations

You can combine multiple transformations by separating them with spaces. For example:

transform: rotate(30deg) scale(1.5) translate(20px, 10px);

This will rotate the element 30 degrees, scale it to 1.5 times its original size, and then translate it 20 pixels to the right and 10 pixels down.
