Links to Resources

Hyperlinks, created with the <a> tag pair, allow redirection to other web pages or documents. The syntax is <a href='url'>link text</a>, with href specifying the address. The target attribute can be set to _blank to open the link in a new tab or _self to open it in the same tab.

Cascading Style Sheets (CSS)

CSS manages the presentation and layout of HTML elements, making it easier to maintain and troubleshoot code. There are three ways to use CSS in HTML:

  • Inline CSS: Applied directly within an HTML tag.
  • Embedded (Internal) CSS: Defined within the <style> tag in the HTML header, affecting all corresponding elements.
  • External CSS: Linked to an external .css file, suitable for large projects.

Scrolling Message

A scrolling message is a visually appealing feature for a website. By defining a container and setting text parameters, a continuous horizontal scrolling effect can be achieved. Keyframes are used to specify the horizontal margins for text movement. This is done within the <head> tag to ensure the effect loads immediately with the webpage.

Adding a Video Clip

To embed a video clip, use the <video> tag, specifying the clip's width and height in pixels. The controls attribute enables play/pause and volume controls. The <source> tag within the video tag specifies the video file and its type. The most commonly supported video format across different browsers is MP4.
